title: "요청에 데이터 넣기"
---

```js
app.use((req, res, next) => {
  req.session.data = "저장하고 싶은 값";
});

app.get("/", (req, res) => {
  console.log(req.session.data);
});
```

대신 세션에 저장하면 다음 요청, 요청마다 세선에 데이터가 저장되어 있다.
한번만 저장하고 다음 요청에는 없었으면 하면 아래 방법으로

```js
app.use((req, res, next) => {
  req.data = "저장하고 싶은 값";
});

app.get("/", (req, res) => {
  console.log(req.data);
});
```

요청이 끝나면 메모리에서 정리된다.
